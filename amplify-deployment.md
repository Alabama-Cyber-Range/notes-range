# AuraNotes - AWS Amplify Deployment

## Build Commands for Amplify

**Build Command:** `npm run build`  
**Output Directory:** `dist`  
**Node Version:** `18` or higher

## Environment Variables (if needed)
```
VITE_APP_NAME=AuraNotes
VITE_MODE=production
```

## Build Configuration

The app will build a static React SPA that works perfectly with Amplify hosting.

## Routing Configuration

Since this is a Single Page Application with React Router, Amplify needs redirect rules.
